Project overview

This project is part of the CSE0001 – Digital Literacy course at VIT Bhopal University. The objective of this project is to develop a strong foundation in digital literacy by exploring essential digital tools, building a professional online presence, and understanding safe and responsible online behavior.

As a Student Digital Ambassador, I completed five structured tasks that focus on different aspects of digital literacy, including creating an awareness infographic, setting up professional profiles, exploring coding and collaboration platforms, practicing email etiquette, and studying cybercrime awareness.

All tasks have been documented and organized in this GitHub repository along with supporting screenshots, links, and a detailed project report. This project reflects both practical learning and personal understanding of digital skills required in academic and professional environments.



Task Summary

Task 1 – Digital Literacy Infographic

Created a one-page infographic using Canva to explain the concept of digital literacy. The design covers important topics such as safe internet practices, useful digital tools for students, and professional online presence. The infographic is included in the `task-1-presentation` folder.



Task 2 – Student Digital Portfolio

Developed a basic professional digital presence by creating and updating profiles on platforms such as GitHub, LinkedIn, and other relevant platforms. A GitHub profile README was also created. Screenshots of all profiles are included in the `task-2-portfolio` folder.




Task 3 – Coding & Collaboration Platforms

Completed a beginner-level coding challenge on a platform like HackerRank to gain exposure to problem-solving. Additionally, created a Google Form titled “Digital Literacy Awareness Quiz” with five questions. Screenshots and proof of completion are available in the `task-3-platforms` folder.



Task 4 – Email Etiquette & Communication

Drafted two professional emails:

1. Requesting an assignment deadline extension
   
3. Applying for a summer internship

Also created a checklist of social media Do’s and Don’ts for responsible online behavior. All files are included in the `task-4-email-etiquette` folder.



Task 5 – Cybercrime Awareness

Researched a cybercrime topic (UPI fraud) and prepared a case study explaining how the crime occurs, who it targets, and its consequences. Additionally, created a “Stay Safe Online” prevention checklist for students. Files are available in the `task-5-cybercrime` folder.

Key learnings

Through this project, I gained a comprehensive understanding of digital literacy and its importance in today’s world. I learned how to create and maintain a professional digital presence using platforms like GitHub and LinkedIn. I also developed basic problem-solving skills by practicing coding on online platforms and understood how collaboration tools like Google Forms can be used effectively.

Additionally, I improved my professional communication skills by drafting formal emails and learning proper email etiquette. The cybercrime awareness task helped me understand common online threats such as phishing and UPI fraud, along with practical steps to stay safe online.

Overall, this project enhanced my confidence in using digital tools responsibly and prepared me for future academic and career opportunities in a digital environment.
